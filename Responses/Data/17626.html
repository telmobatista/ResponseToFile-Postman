Wie Sie in eine weitere Packung desselben Artikels zubuchen.

Rufen Sie den gewünschten Patienten auf. Suchen Sie im unteren Bildteil die Medikation heraus, bei der Sie eine Packung zubuchen möchten. Klicken Sie auf den Button "Packung zubuchen".

Wählen Sie in den Zubuchungsoptionen aus dem Listenfeld die Anzahl der Packungen und klicken auf den Button "zubuchen".

  Im folgenden Schritt können Sie zusätzliche Angaben zur Packung hinterlegen. Das Kontrollfeld "Separat blistern" ist für alle Packungen gültig. Charge, Datum Beipackzettel und Notiz können Sie pro Packung hinterlegen. Mit dem Button "1. Zeile für alle übernehmen" können Sie Ihre Eingaben der 1. Packung auf alle weiteren übernehmen. Mit einem Klick auf den Button "bestätigen" schließen Sie die Eingabe ab.


Sollte der Artikel im Artikelcenter markiert sein als vom Hersteller oder temporr nicht lieferbar oder ist die Lieferbarkeit mit "keine Auswahl getroffen" markiert, kann der Artikel nicht übernommen werden. Es wird Ihnen eine Warnung angezeigt und Sie müssen ein anderes Fertigarzneimittel zur Übernahme auswählen. Siehe auch aCare - Anderes Fertigarzneimittel zubuchen / Rabattvertragswechsel
