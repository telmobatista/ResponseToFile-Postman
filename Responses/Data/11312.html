Medikation ändern / bearbeiten

Vorgehensweise

  • Patienten aufrufen
  • Medikation suchen
  • Button "Medikation ändern"
  • Änderungen der Medikation vornehmen
  • Button "Speichern"

Wie Sie eine bestehende Medikation ändern bzw. bearbeiten.

Rufen den gewünschten Patienten auf und suchen Sie die zu ändernde Medikation. Klicken Sie auf den Button "Medikation ändern".

Nehmen Sie nun Ihre Änderungen vor, wie z.B.:

  • Verordnender Arzt (siehe auch: aCare - Verordnenden Arzt ändern )
  • Medikationsbeschreibung (siehe auch: aCare - Medikationsbeschreibung ändern )
  • Medikationstyp (siehe auch: aCare - Medikationstyp ändern ).

Hinweis: Das Kennzeichen "Separat blistern" kann an dieser Stelle nicht geändert werden, gehen Sie dazu auf den Button "Bestand ändern" (siehe auch: aCare - Kennzeichen "separat blistern" ändern ). 

Mit dem Button "Speichern" bestätigen Sie Ihre Eingaben.

Über das Hauptmenü gelangen Sie zurück zur Startseite.
