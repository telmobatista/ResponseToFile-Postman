Anderes Fertigarzneimittel zubuchen / Rabattvertragswechsel

Vorgehensweise

  • Patienten aufrufen
  • Medikation suchen
  • Button "Packung zubuchen"
  • Arzneimittel und ggf. Packungsanzahl eingeben
  • Button "Suchen"
  • Arzneimittel übernehmen
  • Details eingeben
  • Button "bestätigen"

Wie Sie ein anderes aber vergleichbares Fertigarzneimittel auf eine Medikation zubuchen.

Rufen Sie den gewünschten Patienten auf. Suchen Sie im unteren Bildteil die Medikation heraus, bei der Sie eine Packung zubuchen möchten. Klicken Sie auf den Button "Packung zubuchen".

Tragen Sie in das Eingabefeld den Arzneimittelnamen ein und wählen die Anzahl der Packungen. Bestätigen Sie mit Klick auf den Button "Suchen".

 

 

Im Artikelcenter übernehmen Sie den gewünschten Artikel mit einem Klick auf das Symbol "grüner Pfeil".

 

Alternativ können Sie auch den Strichcode oder den 2D-Code scannen. Scannen Sie den 2D-Code wird die PZN extrahiert und die Suche gestartet. Ist im 2D-Code eine Charge enthalten, merkt sich aCare diese und trägt sie in Schritt 2 automatisch in das entsprechende Feld ein.

Mit der Übernahme werden folgende Artikelkriterien geprüft und verglichen:

  • VPZN/PZN
  • Wirkstoff
  • Wirkstoffmenge
  • Darreichungsform
  • Teilbarkeit
  • Blisterstatus

Stimmen nicht alle Kriterien überein, werden Ihnen die Differenzen in roter Schrift angezeigt. Mit dem Button "Ok" können Sie das Fertigarzneimittel übernehmen.

 

Nach Übernahme des Arzneimittels mit "Ok" oder wenn beide Artikel in allen Kriterien übereinstimmen werden Ihnen weitere Eingabemöglichkeiten zur Eingabe angeboten. Das Kontrollfeld "Separat blistern" ist für alle Packungen gültig. Charge, Datum Beipackzettel und Notiz können Sie pro Packung hinterlegen. Mit dem Button "1.Zeile für alle übernehmen" können Sie Ihre Eingaben der 1. Packung auf alle weiteren übernehmen. Mit einem Klick auf den Button "bestätigen" schließen Sie die Eingabe ab.

Über das Hauptmenü gelangen Sie zurück zur Startseite.
