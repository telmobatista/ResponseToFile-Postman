Dateien in aWebTicket einfügen   

1. Eine Datei zu einem bereits bestehenden Ticket hinzufügen

Vorgehensweise

  • Filter "offene Tickets"
  • Button "Beleg ergänzen" wählen 



  • Button "Datei auswählen" anklicken



  • gewünschte Datei auswählen
  • Button "Öffnen"


  • Datei wurde an aWebTicket angehängt 

 

2. Eine Datei bei Erstellung eines neuen Tickets hinzufügen

Vorgehensweise

  • Button "Neu"
  • neues aWebTicket erstellen, wie in der aWebTicket - Anleitung beschrieben
  • auf den Button "Datei auswählen" klicken

 

  • gewünschte Datei auswählen
  • Button "Öffnen"


  • Datei wurde an aWebTicket angehängt
