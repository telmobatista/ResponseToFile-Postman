Rezeptanforderung für Patient erstellen

Vorgehensweise

  • Hauptmenü "Rezeptanforderung"
  • Anforderungsparameter eingeben
  • Button "absenden"
  • Patientennamen eingeben
  • Button "suchen"
  • Patient markieren
  • Button "Rezeptanforderung erstellen"
  • PDF-Optionen
  • Rezeptanforderung prüfen
  • Button "speichern"

Wie Sie eine Rezeptanforderung für einen Patienten erstellen.

Öffnen Sie das Hauptmenü "Rezeptanforderungen" und geben bei den Anforderungsparameter Ihre gewünschten Parameter ein. Schließen Sie diese Eingabe mit einem Klick auf den Button "absenden" ab.

  • Filtern nach: Wählen Sie "Patient".
  • maximale Reichweite: Wählen Sie das Datum, bis zu welchem die Reichweite der Medikationen enthalten sein soll.
  • Bedarfs- und nichtgeblisterte Dauermedikation: Markieren Sie das entsprechende Kontrollfeld, wenn diese Medikationstypen mit in die Rezeptanforderung einfließen sollen.
  • alle Anforderungen einfügen: Die Auswahl "nein" erstellt eine neue Rezeptanforderung nach den gewählten Parametern. Die Auswahl "ja" bewirkt, dass alle Anforderungen, die sich im Quermenü "offene Rezeptanforderungen" befinden, gelöscht werden und in die neue Anforderung mit aufgenommen werden.

Erfassen Sie im Eingabefeld den Namen des gewünschten Patienten und Klicken auf den Button "Suchen".

  

Markieren Sie im Optionsfeld den entsprechenden Patienten und Klicken auf den Button "Rezeptanforderung erstellen".

Gibt es für die gewählten Parameter keine betroffenen Medikationen wird Ihnen ein Hinweis angezeigt.

Andernfalls wird eine Liste der betroffenen Medikationen erstellt. Sie können diverse Einstellungen vornehmen.

  • Anforderung zum: Wählen Sie das Datum zu wann die Rezeptanforderung gelten soll.
  • Absendertext: Wählen Sie über das Optionsfeld, ob die Apotheke oder ein freier Text als Absender der Rezeptanforderung auftritt.
  • Kopftext: Text, der auf die Rezeptanforderung aufgebracht wird. Soll der Text auf allen Rezeptanforderungen vorbelegt sein, klicken Sie auf den Button "Kopftext speichern".
  • Dosierzeiten: Sollen die Dosierzeiten auf der Rezeptanforderung ausgedruckt werden, markieren Sie das Kontrollfeld.
  • PZN/Arzneimittelname: Diese Angaben können Sie durch überschreiben anpassen.
  • Individuelle Information: Hier können Sie eine Information für den Arzt hinterlegen.
  • in der Rezeptanforderung ignorieren: Aktivieren Sie das Kontrollfeld, wenn die Medikation nicht auf dieser Rezeptanforderung aufgebracht werden soll.

Mit einem Klick auf den Button "speichern" wird die Rezeptanforderung erstellt und im Quermenü " offene Rezeptanforderungen" abgelegt.

Über das Hauptmenü gelangen Sie zurück zur Startseite.
