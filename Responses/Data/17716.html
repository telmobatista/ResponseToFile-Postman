aCare - gesamte Rezeptanforderung drucken

Vorgehensweise

  • Hauptmenü "Rezeptanforderung"
  • Quermenü "Offene Rezeptanforderung"
  • Symbol "PDF für gesamte Rezeptanforderung öffnen/drucken"
  • Optionsfeld "Öffnen mit" wählen
  • Button "OK"
  • PDF ausdrucken

Wie Sie eine gesamte Rezeptanforderung drucken.

Öffnen Sie das Hauptmenü "Rezeptanforderungen" und wechseln in das Quermenü "offene Rezeptanforderungen". Klicken Sie in der gewünschten Anforderung auf das Symbol "PDF für gesamte Rezeptanforderung öffnen/drucken". 

Im erscheinenden Dialog können Sie die Datei öffnen oder speichern. Bestätigen Sie Ihre Auswahl mit dem Button "OK".

Mit der Auswahl "Öffnen mit" wird Ihnen ein PDF geöffnet, das Sie dann ausdrucken können.

Mit der Auswahl "Datei speichern" können Sie die Datei an einem gewünschten Ort speichern und von dort weiter bearbeiten.

Über das Hauptmenü gelangen Sie zurück zur Startseite.

Voraussetzung ist, dass Sie bereits eine Rezeptanforderung erstellt haben.
