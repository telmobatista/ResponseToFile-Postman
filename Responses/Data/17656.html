Offene Rezeptanforderung als erhalten markieren

Vorgehensweise

  • Hauptmenü "Rezeptanforderung"
  • Quermenü "Offene Rezeptanforderung"
  • Symbol "Anforderung komplett erhalten"

Wie Sie eine Rezeptanforderung als komplett erhalten markieren.

Öffnen Sie das Hauptmenü "Rezeptanforderungen" und wechseln in das Quermenü "offene Rezeptanforderungen". Klicken Sie in der gewünschten Anforderung auf das Symbol "Anforderung komplett erhalten". 

Die Anforderungen bleiben so lange in dieser Liste, bis Sie diese entfernen oder bis Sie bei allen beinhalteten Medikationen eine Packung zubuchen.

Über das Hauptmenü gelangen Sie zurück zur Startseite.

Voraussetzung ist, dass Sie bereits eine Rezeptanforderung erstellt haben.
