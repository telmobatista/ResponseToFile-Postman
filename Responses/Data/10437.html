Zeitsteuerung eines Makros

Sie können dem Makro ein Startdatum und eine Startuhrzeit zuordnen. Das Makro wird dann zu diesem Zeitpunkt automatisch aktiviert.

  • Blenden Sie die Zusatzfunktionen des Makro-Rekorders ein
  • Sie haben folgende Möglichkeiten, die Zeitsteuerung zu regeln:

 


Zt

Steht für Zeitpunkt. Geben Sie ein Startdatum ein. Das Makro wird einmalig an dem eingegebenem Tag gestartet. 

Ordnen Sie danach eine Uhrzeit zu.

Tä

Steht für tägliche Zeitsteuerung. Geben Sie eine Periode ein: Jeden 1 Tag. D.h. das Makro wird jeden Tag gestartet. Jeden 2 Tag. D.h das Makro wird jeden 2 Tag gestartet usw.

Ordnen Sie danach eine Uhrzeit zu.

Wö

Steht für wöchentliche Zeitsteuerung. Geben Sie eine Periode ein (jede 1 oder 2 usw. Woche) und den/die Tag/Tage auswählen/markieren (mo/di/mi/do/fr/sa/so).

Ordnen Sie danach eine Uhrzeit zu.

mo

Steht für monatliche Zeitsteuerung. Geben Sie eine Periode ein (jeder 1 oder 2 usw. Monat). Danach den Tag festlegen, an dem das Makro gestartet werden soll.

Ordnen Sie danach eine Uhrzeit zu.

ja

Steht für jährliche Zeitsteuerung. Geben Sie eine Periode ein (jedes 1 oder 2 usw. usw. Jahr). Danach den Monat und den Tag eingeben.

Ordnen Sie danach eine Uhrzeit zu.
