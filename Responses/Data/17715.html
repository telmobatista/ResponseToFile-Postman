Rezeptanforderung pro Arzt als Datei herunterladen

Vorgehensweise

  • Hauptmenü "Rezeptanforderung"
  • Quermenü "Offene Rezeptanforderung"
  • Symbol "Anforderungen pro Arzt als ZIP-Datei herunterladen"
  • Datei speichern
  • Button "OK"
  • Weiterbearbeiten der Datei wie gewünscht

Wie Sie die Rezeptanforderung pro Arzt als ZIP-Datei zur weiteren Verarbeitung herunterladen.

Öffnen Sie das Hauptmenü "Rezeptanforderungen" und wechseln in das Quermenü "offene Rezeptanforderungen". Klicken Sie in der gewünschten Anforderung auf das Symbol "Anforderungen pro Arzt als ZIP-Datei herunterladen". 

Im erscheinenden Dialog können Sie die Datei öffnen oder speichern. Bestätigen Sie Ihre Auswahl mit dem Button "OK".

Nach dem Speichern können Sie die Datei wie gewünscht weiter verarbeiten.

Über das Hauptmenü gelangen Sie zurück zur Startseite.

Voraussetzung ist, dass Sie bereits eine Rezeptanforderung erstellt haben.
