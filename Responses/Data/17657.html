Rezeptanforderung pro Arzt drucken

Vorgehensweise

  • Hauptmenü "Rezeptanforderung"
  • Quermenü "Offene Rezeptanforderung"
  • Symbol "Anforderungen pro Arzt"
  • Symbol "Rezeptanforderung als PDF erstellen"
  • PDF ausdrucken

Wie Sie ein separates PDF einer Rezeptanforderung pro Arzt drucken.

Öffnen Sie das Hauptmenü "Rezeptanforderungen" und wechseln in das Quermenü "offene Rezeptanforderungen". Klicken Sie in der gewünschten Anforderung auf das Symbol "Anforderungen pro Arzt". 

Unterhalb der Rezeptanforderung erscheint jeder Arzt, der in der Anforderung enthalten ist. Mit einem Klick auf das Symbol "Rezeptanforderung als PDF erstellen" hinter dem Arzt wird ein PDF für diesen Arzt erstellt und kann ausgedruckt werden.

Über das Hauptmenü gelangen Sie zurück zur Startseite.

Voraussetzung ist, dass Sie bereits eine Rezeptanforderung erstellt haben.
