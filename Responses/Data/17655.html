offene Rezeptanforderung im Detail ansehen/bearbeiten

Vorgehensweise

  • Hauptmenü "Rezeptanforderung"
  • Quermenü "Offene Rezeptanforderung"
  • Symbol "Anforderung ansehen"
  • optional Symbol "Position erhalten"
  • optional Symbol "PDF Liste drucken"
  • optional Symbol "Excel Liste exportieren"

Wie Sie offene Rezeptanforderungen ansehen und bearbeiten.

Öffnen Sie das Hauptmenü "Rezeptanforderungen" und wechseln in das Quermenü "offene Rezeptanforderungen". Klicken Sie in der gewünschten Anforderung auf das Symbol "Anforderung ansehen". 

Die Rezeptanforderung im Detail zeigt Ihnen die Artikel an, für die Rezepte noch ausstehend sind. Mit einem Klick auf das Symbol "Position erhalten" wird die Zeile entfernt. Dies dient der Empfangskontrolle. Es wird keine Packung bei der Medikation zugebucht.

Die Anforderungen bleiben so lange in dieser Liste, bis Sie diese entfernen oder bis Sie bei der dazugehörenden Medikation eine Packung zubuchen.

Mit einem Klick auf das Symbol "PDF Liste drucken" erstellen Sie eine Rezeptanforderung der Artikel, die sich noch in der Rezeptanforderung befinden.

Mit einem Klick auf das Symbol "Excel Liste exportieren" erstellen Sie eine Excel Auflistung über die Artikel, die sich noch in der Rezeptabforderung befinden.

Über das Hauptmenü gelangen Sie zurück zur Startseite.

Voraussetzung ist, dass Sie bereits eine Rezeptanforderung erstellt haben.
