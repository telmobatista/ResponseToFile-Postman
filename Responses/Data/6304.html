Mehrere Fingerabdrücke hinterlegen

  • Öffnen Sie die Mitarbeiterverwaltung unter "Konfigurations-Center/ Mitarbeiterverwaltung"
  • Wählen Sie den Mitarbeiter aus, für den Sie weitere Fingerabdrücke hinterlegen möchten.
  • Über den Button "Ändern" hinter  dem Feld "Fingerscan" wird eine Maske zur Verwaltung der Fingerabdrücke geöffnet.

Sofern vorhanden, wird ein bereits vorhandener Scan eines Fingerabdrucks angezeigt. Über den Button "Neu" können die Scans weiterer Fingerabdrücke hinzugefügt werden.
