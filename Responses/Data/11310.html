Dosierung ändern

Vorgehensweise

  • Patienten aufrufen
  • Medikation suchen
  • Button "Medikation ändern"
  • Symbol "Bearbeiten"
  • Änderungen vornehmen
  • Button "Speichern"

Wie Sie die aktuelle Dosierung einer Medikation ändern.

Rufen den gewünschten Patienten auf und suchen Sie die Medikation, an der Sie die Dosierung ändern möchten. Klicken Sie auf den Button  "Medikation ändern".

Wählen Sie das Symbol "Bearbeiten".

Die Dosierungsdetails werden angezeigt. Im oberen Teil können Sie Beschreibung, Gültigkeit und Applikationsrhythmus, im unteren Teil die Dosierung und Hinweise angepassen. 

Hinweis: Standardmäßig ist eine Änderung der Dosierung nur möglich, wenn Sie am gleichen Tag wie die Neuanlage dieser Dosierung erfolgt. Sobald der Dosierungsbeginn in der Vergangenheit liegt, ist eine Änderung der Dosierung nicht mehr möglich. Erfassen Sie in diesem Fall eine neue Dosierung (siehe auch: aCare - Neue Dosierung anlegen ).

Bestätigen Sie die Änderungen mit dem Button "Speichern".

Hierbei wird der Patient gesperrt (siehe auch: aCare - Freigabe von Patienten ).

Über das Hauptmenü gelangen Sie zurück zur Startseite.
